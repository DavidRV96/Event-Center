## 📖 Introduction
The **Event Center Website** is a front-end project designed to simulate a real-world venue website. It includes multiple pages that introduce the venue, highlight upcoming events, and provide users with a way to inquire or book events.  

This project focuses on:
- Clean, semantic HTML
- External CSS styling
- Responsive design principles
- User-friendly navigation and layout

It is suitable for portfolio presentation and demonstrates mastery of HTML and CSS fundamentals.

---

## ✨ Features
✅ **Multi-page website**
- `index.html` – Home page with an overview of the event center and featured content  
- `about.html` – Background information, mission, and venue details  
- `events.html` – Event listings and schedule  

✅ **Semantic HTML Structure**
- Uses `header`, `nav`, `main`, `section`, `article`, and `footer` elements

✅ **Navigation Menu**
- Fully linked navigation across all pages

✅ **Content Elements**
- Headings (`h1–h6`), paragraphs, and lists
- Tables for event schedules or pricing
- Images in multiple formats (`.jpg`, `.png`, `.gif`) with descriptive `alt` text

✅ **Forms & Interactivity**
- Contact form with text input, email input, textarea, and submit button
- Hover and focus effects for links and buttons

✅ **CSS Styling**
- External CSS file
- Consistent color scheme and typography
- Flexbox layout for structure and alignment
- Box model usage (padding, margin, borders)
- Custom button styles
- Class and ID selectors

✅ **Responsive Design**
- Layout adapts to both mobile and desktop screen sizes

✅ **Footer**
- Copyright and social media links
